Create ConfigLoader

Path: commons/src/main/java/com/osrs/commons/ConfigLoader.java

Content:

java
package com.osrs.commons;
import java.nio.file.*;
import java.util.Map;
import com.fasterxml.jackson.databind.ObjectMapper;
public class ConfigLoader {
  public static Map<String,Object> load(Path p) throws Exception {
    ObjectMapper m = new ObjectMapper();
    return m.readValue(Files.readAllBytes(p), Map.class);
  }
}
Add dependency: implementation("com.fasterxml.jackson.core:jackson-databind:2.15.2") in commons/build.gradle.kts.

Acceptance: ./gradlew :commons:test passes a simple test.

Add SafeSleep helper

Path: commons/src/main/java/com/osrs/commons/SafeSleep.java

Content:

java
package com.osrs.commons;
public final class SafeSleep {
  public static void ms(long ms) {
    try { Thread.sleep(ms); } catch (InterruptedException ignored) { Thread.currentThread().interrupt(); }
  }
}
Acceptance: Compiles.
